# semiscope receipts

The public, append-only track record of the semiscope engine.

- `ledger.json` — every flagship prediction with interval, timestamp, and
  a sha256 commitment. Scored against primary sources (SEC filings, MOPS
  prints, EIA sheets) when actuals land. Misses stay forever.
- `mass-ledger-summary.json` — calibration stats for ~1,900 monthly
  Taiwan supply-chain predictions + hash commitments for the open month.
- `differentials.json` — where we disagree with street consensus (delta
  only), and who won once resolved.
- `index.html` — human-readable view, worst calls first.

**Verification:** each prediction's `commitment` =
`sha256(target|point|low|high|made_at)[:16]`. The git history of this repo
timestamps every addition. If a row ever changes after insertion, the diff
is public.

Generated by the semiscope engine (private repo, git 1c9b993).
